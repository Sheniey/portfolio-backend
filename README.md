
## Instalation
### Linux / MacOS

~~~bash
python3 -m venv venv
~~~

~~~bash
source venv/bin/activate
~~~

~~~bash
pip install -r requirements.txt
~~~

~~~bash
fastapi dev app/main.py
~~~

### Windows
~~~bash
python -m venv venv
~~~

~~~bash
venv\Scripts\activate
~~~

~~~bash
pip install -r requirements.txt
~~~

~~~bash
fastapi dev app/main.py
~~~
